# Job Scanner

## Installation

1. Open **Chrome**
2. Go to: `chrome://extensions/`
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked**
5. Select the `linkedin-job-scanner` folder
6. The extension icon will appear in your toolbar

---

## How to Use

1. Go to LinkedIn Jobs and run a search  
   Example:  linkedin.com/jobs/search?keywords=developer

2. Click the extension icon

3. Type a keyword to filter by  
- It searches inside each job description  
- Leave blank to collect all jobs  

4. Set the delay between jobs  
- Use the slider or number input  
- Range: **500ms – 10s**  
- Higher delay is safer to avoid rate limiting  

5. Click ▶ **START SCAN**  
- The extension will click through each job one by one  

6. Matched jobs will appear in the results list  
- Click any job to open it  

7. Click ⬇ **DOWNLOAD RESULTS (.txt)**  
- Saves all matched job links to a file
